# 自主禁言插件

允许用户主动禁言自己或解除自己的禁言。

## 功能介绍

- 用户可以主动禁言自己指定的时间
- 支持群聊和私聊操作
- 智能判断机器人权限，必要时请求群主协助
- 支持解除禁言功能

## 使用方法

### 在群聊中使用

#### 禁言自己指定分钟数

```bash
/ban 30          # 禁言自己 30 分钟
/禁言 30         # 使用中文别名
```

#### 解除禁言

```bash
/ban 0           # 解除自己的禁言
/禁言 0          # 使用中文别名
```

### 在私聊中使用

在私聊中使用时，需要额外提供群号：

```bash
/ban 30 123456789    # 在群 123456789 中禁言自己 30 分钟
/ban 0 123456789     # 在群 123456789 中解除自己的禁言
```

## 配置说明

无需特殊配置。机器人需要在群中具有管理员权限才能直接执行禁言操作，否则会请求群主协助。
